 2. Get All Courses

 Method: GET
 URL: `http://localhost:5000/api/courses/courses`
 Authorization: Bearer Token (optional if public)
 Expected Response: Array of all courses.

3. Get Teacher‚Äôs Courses

 Method: GET
 URL: `http://localhost:5000/api/courses/my-courses`
 Authorization: Bearer Token of teacher
 Expected Response: Array of courses created by the logged-in teacher.

4. Update Course

 Method: PUT
 URL: `http://localhost:5000/api/courses/course/:courseId`
 Replace `:courseId` with the actual `_id` of the course.
 Authorization: Bearer Token of teacher
 Body (raw JSON):
{
  "courseTitle": "Intro to AI Updated",
  "totalPriceOfCourse": "110"
}
Expected Response: Updated course object.

5. Delete Course

 Method: DELETE
 URL: `http://localhost:5000/api/courses/course/:courseId`
 Authorization: Bearer Token of teacher
 Expected Response: Success message, course removed.

### 1Ô∏è‚É£ Add Learning Outcomes

**Method:** `POST`
**URL:** `http://localhost:5000/api/courses/:courseId/learning-outcome`

**Headers:**

```
Key: Authorization
Value: Bearer <YOUR_JWT_TOKEN>
```

**Body (raw JSON):**

```json
{
  "learningOutcomeOfCourse": [
    "Understand the basics of AI",
    "Build a simple AI model",
    "Deploy AI application"
  ]
}
```

**Notes:**

* Replace `:courseId` with the actual course `_id` from your database.
* Make sure `learningOutcomeOfCourse` is an **array**.
